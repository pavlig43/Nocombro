# Система фасовки специй

## Задача

Создать автоматическую систему для расфасовки специй по весу (например, 300 грамм в пакет).

## Общая схема

```
         ┌─────────────────┐
         │   Конусный      │
         │   бункер        │
         └───────┬─────────┘
                 │
         [Slide gate заслонка]
                 │
         [Линейный актюатор]
                 │
    ┌────────────┴────────────┐
    │                         │
[Пакет на весах]         [Arduino]
    │                         │
[Тензодатчик]           [Кнопки]
    │                    100 200 300 500
[Модуль HX711]              СТАРТ СТОП
    │
[Arduino - читает вес]
```

## Логика работы

1. **Выбор веса:** Пользователь нажимает кнопку с нужным весом (100/200/300/500 г)
2. **Подготовка:** Пакет ставится на весы (платформу с тензодатчиком)
3. **Старт:** Нажимается кнопка СТАРТ
4. **Дозирование:**
   - Arduino открывает заслонку (актюатор выдвигается)
   - Специи сыпутся под действием гравитации
   - Тензодатчик через HX711 передаёт реальный вес в Arduino
5. **Остановка:** При достижении заданного веса Arduino закрывает заслонку
6. **Повтор:** Готово для следующего пакета

> **Кнопка СТОП** — для аварийной остановки процесса

## Конструкция бункера

### Тип
- **Конический бункер** (сварить из нержавейки самостоятельно)
- Гравитационная подача — щнек не нужен!

### Заслонка
- **Slide Gate** (скользящая заслонка)
- Пластина сдвигается влево-вправо
- Привод: линейный актюатор 12V

## Подключение к Arduino

### Тензодатчик (весы)
```
Тензодатчик ──> HX711 ──> Arduino
  (красный,     (DT, SCK)    A0, A1
   чёрный,
   белый,
   зелёный)
```

### Актюатор
```
Arduino (Digital Pin) ──> Реле ──> Актюатор 12V
                                     └── Блок питания 12V
```

### Кнопки
```
Кнопки ──> Arduino (Digital Pins 2-6)
         └── Подтягивающие резисторы или INPUT_PULLUP
```
